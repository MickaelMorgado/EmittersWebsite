# Emitters - Free Developer Tools & Web Apps

![Next.js](https://img.shields.io/badge/Next.js-15-black)
![React](https://img.shields.io/badge/React-18-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)

Collection of free online developer tools, utilities, and web applications. Built with Next.js, React, Three.js, and modern web technologies.

## Featured Tools

### Trading & Productivity
- **PNL Calendar** - Free trading journal with profit/loss tracking and visualization
- **Daily Todo Tracker** - Habit tracking with progress charts

### AI & Voice
- **MemoGPT** - AI-powered prompt manager for ChatGPT
- **PC AI Assistant** - Voice-activated AI chat with galaxy visualization
- **TikTok TTS** - Text-to-speech for TikTok live streams

### 3D & Graphics
- **3D CAD** - Browser-based 3D modeling tool with STL/OBJ export
- **G-Code Timelapse** - 3D print visualization from G-code files
- **Camera Effects** - Real-time webcam filters and effects

### Development Utilities
- **Image Compressor** - Bulk compress images to target file size
- **3D Printer Monitor** - Multi-camera monitoring for 3D prints
- **Sounder** - Audio sampler and music production tool
- **EMF Detector** - Electromagnetic field simulator

### Gaming
- **S.T.A.L.K.E.R. 2 Ammo Tracker** - Inventory and ammo management for S.T.A.L.K.E.R. 2

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 18, TypeScript, Tailwind CSS
- **3D:** Three.js, React Three Fiber, Drei
- **Post-processing:** React Three Postprocessing
- **State:** Zustand
- **Database:** Supabase

## Getting Started

```bash
# Install dependencies
cd node-projects/my-app
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

Deploy to Vercel with zero configuration:

```bash
vercel --prod
```
